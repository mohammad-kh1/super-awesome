## Contributing

Found a missing awesome list? Want to improve the design?

1. **Fork** the repo
2. Add your list (with category & short description)
3. Open a **Pull Request**

**Guidelines:**
- Only **active, well-maintained** awesome lists
- One entry per repo
- Use consistent formatting
- Prefer lists with **1k+ stars** and recent updates
